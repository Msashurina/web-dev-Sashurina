1. Research for and describe the ways to open developers console in Mozilla Firefox and Safari.

Opening Developer Console in Mozilla Firefox:

- Method 1: Pressing the keyboard shortcut "Ctrl + Shift + I" (or "Cmd + Opt + I" on macOS) will open the Developer Tools panel, where you can find the Console tab.

- Method 2: Go to the menu by clicking the three horizontal lines in the top-right corner, then select "Web Developer" and click on "Web Console".

Opening Developer Console in Safari:

- Method 1: Pressing the keyboard shortcut "Option + Cmd + C" will open the Developer Tools panel, where you can find the Console tab.

- Method 2: Go to the menu by clicking "Develop", then select "Show Error Console".

2. What way of including internal links into the HTML page you would consider generally preferable: absolute or relative links? Describe use cases for choosing absolute links and choosing relative links.

Generally, it is preferable to use relative links over absolute links for internal links:

- Relative links: These links are relative to the current page's location within the directory structure. They are shorter and more flexible, making them easier to manage and maintain. Relative links are commonly used when linking to other pages within the same website.

- Absolute links: links include the full URL, including the protocol (e.g., http://) and domain. Absolute links are necessary when linking to external websites or when the page structure might change.

Use cases for choosing absolute links:

- Linking to other websites or web pages outside of your domain.

- When the page structure might change frequently, and you want to ensure the link remains intact.

Use cases for choosing relative links:

- Linking to other pages within the same website.

- When the website structure is unlikely to change, making it easier to manage and maintain links.

- When the page is hosted locally and not yet deployed to a live server.

3. Research and write about image formats used in web-development. What is the difference between JPG, GIF and PNG formats? What is WEBP image type? What is SVG?

- JPG (or JPEG): It is a lossy compression format commonly used for photographs or complex images with many colors. JPGs support millions of colors and smaller file sizes, but they sacrifice some image quality.

- GIF: It is a lossless compression format commonly used for simple animations and graphics with limited colors. GIFs support transparency and animation but have a limited color palette.

- PNG: It is a lossless compression format commonly used for images with transparency support. PNGs preserve image quality but often have larger file sizes compared to JPGs.

- WEBP: It is a newer image format created by Google. WEBP supports both lossy and lossless compression and generally provides smaller file sizes while maintaining good image quality. It is supported by modern browsers but may not be compatible with older browsers.

- SVG: It stands for Scalable Vector Graphics. SVG is a vector image format that uses XML-based markup to describe graphics. Unlike raster images (JPG, GIF, PNG), SVGs can be scaled up or down without losing quality. They are well-suited for logos, icons, and other graphics that require flexibility in size and resolution. SVGs can be animated and modified using CSS and JavaScript as well.
